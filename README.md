# cli
GOTRIX command line helper.

## Installation
```
go get -u github.com/gotrix/cli/...
```

## Usage
```
Usage of gotrix:
  -no-color
    	do not colorize output
  -path string
    	path to use
  -quiet
    	do not print any output
```

## Commands
List of available cli commands.
